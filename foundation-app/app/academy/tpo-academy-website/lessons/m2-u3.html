Universal Post Anatomy

10 minutes TPO Install

Every post — regardless of mode — runs through the same three-part architecture. Each component has exactly one job.

ComponentIts One JobThe Standard
HookStop the scroll in 1.5 secondsPattern interrupt, instant relatability, or a question that stings. No warm-up. Start mid-point.
MessageDeliver exactly what the hook promisedOne mode only. If you cannot classify it in 5 seconds, it is mixing modes and is not ready.
CTADirect the next move with zero ambiguityOne specific action. "Comment AUDIT for a free review." Vague CTAs are invisible CTAs.

When all three jobs are done well in sequence, you have a post that stops the scroll, holds attention, and produces a measurable next action. When any one fails, the entire post fails — a great hook with a weak CTA produces views but no clients.
